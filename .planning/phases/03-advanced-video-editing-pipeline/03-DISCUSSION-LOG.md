## Areas Discussed

### 1. Karaoke-Style Subtitles (TikTok Style)
- **Choice:** Generate Advanced SubStation Alpha (.ass) files.
- **Mechanism:** Compute word-by-word timestamps inside chunks and apply `{\kf<centiseconds>}` karaoke tags. The active word changes color (e.g., to Yellow/Green) as it is spoken.
- **Integration:** The style (color, font size) will load from the user's Supabase profile settings.

### 2. Auto-Zoom / Jump Cuts
- **Mechanism:** Gemini will provide the hook sentence. The backend matches the hook text to subtitle segments, identifies the timestamps, and overlays a 15% zoom-in crop filter during those exact seconds.

### 3. Silence/Pause Removal
- **Mechanism:** Run FFmpeg `silencedetect` on the source clip. Parse the logs to find gaps where volume is below -40dB for > 1.0 second. Use an FFmpeg `trim`/`concat` filter complex to dynamically skip these silences while preserving AV synchronization.
